# Indsro - Industrial Website

Indsro is a responsive, multipage industrial website built using Bootstrap, HTML, CSS, and JavaScript. This website is designed to be mobile-friendly and fully responsive across different devices and screen sizes.

## Features

- Fully responsive design for mobile, tablet, and desktop screens
- Multipage structure (Home, About, Services, Contact, etc.)
- Smooth animations and transitions
- Clean and modern UI design
- Cross-browser compatibility
- Built with Bootstrap for faster development and responsiveness

## Technologies Used

- HTML5
- CSS3 (with Bootstrap 5)
- JavaScript (Vanilla JS)
- FontAwesome Icons
- Google Fonts
